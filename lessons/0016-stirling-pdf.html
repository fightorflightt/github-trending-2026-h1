一句话说清楚

Stirling-PDF 是 GitHub 上排名第一的 PDF 处理工具。合并、拆分、旋转、压缩、格式转换、加水印、电子签名……你能想到的 PDF 操作，它全包了。最关键的：所有处理都在你自己的设备上完成，PDF 不上传任何云端。

能干什么？

  • 合并/拆分 PDF 文件
  • PDF ↔ 图片/Word/PPT/Excel 互转
  • 压缩 PDF 体积
  • 添加/删除 水印、密码
  • 电子签名
  • OCR（扫描件识别文字）

为什么火？

  1. 「做了件小事，但做到了极致」——PDF 处理人人都需要，但没人想装 Adobe Acrobat
  2. 隐私是刚需：合同、证件、病历……这些 PDF 谁敢上传到陌生网站？
  3. 部署简单：一行 Docker 命令就能跑起来

💡 做独立开发的启发

  • 「把一件小事做到极致」是最适合独立开发的策略：PDF 工具看起来不起眼，84K 星说明需求巨大
  • 巨头留下的缝隙都是机会：Adobe Acrobat 又贵又笨重 → Stirling-PDF 免费又轻量
  • 「隐私优先」能让你的小工具产生溢价：同样做 PDF，Stirling 跟在线工具的核心差异就是「不上传」
  • 一个实用工具项目可以活很久：不追热点，做一个解决真实问题的东西，用户会一直用
