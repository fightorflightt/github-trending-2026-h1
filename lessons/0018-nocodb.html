📊 第 18 课：NocoDB

开源版 Airtable——用 Excel 的方式管理数据库，不用写一行代码

📊 无代码 🗄️ 数据库 ⭐ 63K+ Airtable 替代

一句话说清楚

NocoDB 是 Airtable 的开源替代品。它的核心体验很简单：连接一个数据库（MySQL、PostgreSQL 等），然后用电子表格的界面来管理数据。你看到的是 Excel，但它背后是真的数据库。不用学 SQL，也能做增删改查、筛选排序、协作编辑。

解决了什么问题？

😫 以前

想看数据库里存了什么 → 得会写 SQL 查询 → 结果是一堆看不懂的表格 → 改数据还得写 SQL

😎 用 NocoDB

打开网页 → 看到的就是电子表格 → 像操作 Excel 一样增删改 → 自动同步到数据库

为什么火？

  1. Airtable 太贵了——团队用一用就上千美元/月，NocoDB 自部署免费
  2. 「电子表格」是全世界最通用的界面——不需要培训，谁都会用
  3. 开源 + 不锁定数据——数据永远在你自己的数据库里

💡 做独立开发的启发

  • 「把已有的东西换个更友好的界面」就是好产品：数据库存在几十年了，NocoDB 只做了一件事——把界面换成电子表格
  • 独立开发可以瞄准「大产品的开源替代品」：Airtable → NocoDB，Zapier → n8n，Google Analytics → Plausible……这个清单可以一直列下去
  • 「电子表格界面」是 toB 产品的万能钥匙：你的企业用户可能不懂任何软件，但一定懂 Excel
