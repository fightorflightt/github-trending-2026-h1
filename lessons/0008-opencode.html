一句话说清楚

opencode 跟第 3 课的 Claude Code 做的是同一件事——AI 帮你在终端里写代码。但区别在于：它是完全开源、不绑定任何大模型的。你可以用 OpenAI、Claude、本地模型……随便换，不被任何一家公司锁定。

为什么火？

  1. 「不被锁定」是开发者最在意的：不想押注单一 AI 公司
  2. 开源社区的力量：几十个贡献者在维护，迭代速度极快
  3. 灵活度高：可以接入本地模型，完全离线使用

💡 做独立开发的启发

  • 「不绑定平台」是用户的核心诉求：做产品时，让用户能自由导入导出数据、自由选择搭配
  • 「开源替代品」永远有机会：大厂出什么，开源社区就做什么，而且更灵活、更自由
  • 对独立开发者的实用价值：如果你想做 AI + 编程类的独立产品，opencode 是比 Claude Code 更好的底座（开源、免费、灵活）
