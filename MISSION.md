# Mission

## What I'm learning
2026 年上半年热门 GitHub 开源项目。

## Why I'm learning it
我是产品经理，也在做独立开发。我需要通过热门 GitHub 项目了解当前市场的技术趋势和关注方向，为自己做独立产品找到方向和灵感。

## How I'll use it
- 了解每个热门项目是干嘛的、怎么用、解决什么问题
- 从中发现独立开发的产品机会
- 开阔技术视野，跟上行业动态

## Success criteria
- 完成 2026 H1 热门 GitHub 项目系列课程（共 20 节）
- 每节课能用自己的话说清楚：这个项目做什么、怎么用、为什么火
- 从中提炼出至少 3 个对自己独立开发有启发的方向

## Constraints
- 用户不懂代码，讲解必须用大白话
- 偏教程向：项目干嘛的 + 怎么用，而非深度产品分析
- 每节课一个项目，短小精悍
